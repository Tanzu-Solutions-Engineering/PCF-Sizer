Notes for vSphere

For more details on the minimum starting requirements refer to the documentation

Your vSphere cluster must have the following resources available before you install Tanzu Application Service

  • Two public IP addresses: One for Tanzu Application Service and one for Ops Manager
  • Ops Manager Director must have HTTPS access to vCenter and ESX hosts on TCP ports 443, 902, and 903.
  • For specific IaaS resource requirements, see below for more detail
