Notes for Azure

For more details on the minimum starting requirements refer to the documentation

Your Azure account must have the following resources before you install Tanzu Application Service

  • Pay-as-you-go subscription on your Azure account.
  • 1 Storage Account
  • Sufficiently high quota limits (Recommended: 100 cores) on your Azure account.
  • Installing Tanzu Application Service requires more than the default 20 cores
  • 3 Public IPs
  • 1 “jumpbox” VM that runs the deployment using BOSH
